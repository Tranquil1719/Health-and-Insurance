# Health-and-Insurance Dataset
This project focuses on the analysis and visualization of a comprehensive medical insurance dataset to uncover key insights related to demographics, health risk factors, healthcare utilization, and financial performance. The dataset contains detailed information on individuals, including age, sex, and their health status offering deep insight on their health and insurance pattern.
